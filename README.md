# research-data-library-year-intervals
Research data for the study on librarian-assigned and predicted year intervals.

# Data and Code for Evaluating LLM-based temporal subject assignment in fiction: the impact of evaluation frameworks on metadata interpretation  

This repository contains the data and code associated with the article:
Evaluating LLM-based temporal subject assignment in fiction: the impact of evaluation frameworks on metadata interpretation  
Fereshta Westin and Johan Eklund  
Submitted to *Cataloging & Classification Quarterly*

## Overview

This repository includes the processed data, model outputs, and analysis scripts used in the study. The study examines the automatic prediction and evaluation of temporal information, specifically year intervals or time periods associated with fictional works.

The materials are provided to support transparency and reproducibility. The repository does not include full-text copyrighted source material. Instead, it contains processed outputs, derived data, and code used for the analyses reported in the article.


## File and Folder Descriptions
| File or folder | Description |
|---|---|
| `LLM_outputs/` | Folder containing outputs generated by the large language model. These files represent the model-produced temporal predictions used in the analysis. |
| `data/` | Folder containing processed and derived data used in the evaluation. |
| `data/fiction novels.xlsx` | The `fiction novels` folder contains the original set of 72 novels before any filtering was applied. |
| `scripts/` | Folder containing Python scripts used for analysis. |
| `scripts/F1_comparison.py` | Python script used to calculate or compare F1-based evaluation results. |
| `scripts/RMSE_comparison.py` | Python script used to calculate or compare RMSE-based evaluation results. |
| `data/RMSE_vs_f1.xlsx` | Spreadsheet containing comparison results between RMSE and F1-based evaluation. |
| `data/Year_outputs.xlsx` | Spreadsheet containing year-related outputs used in the evaluation. |
| `data/libris_searchstrings.txt` | Text file with LIBRIS search strings to obtain novels with time periods |



## Data Description

The data in this repository consists of processed and derived outputs used to evaluate automatically generated temporal predictions. The dataset includes year-related outputs and evaluation results used to compare different ways of assessing predicted temporal information.

The repository does not contain full-text copyrighted material. Where copyrighted or restricted source material was used in the study, only derived outputs and evaluation data are included.

## Code Description

The repository includes two Python scripts:

- `F1_comparison.py`, which is used for F1-based comparison.
- `RMSE_comparison.py`, which is used for RMSE-based comparison.

These scripts support the analysis reported in the article by calculating and comparing evaluation results for temporal predictions.

## Software and Requirements

- `pandas`
- `openpyxl`
